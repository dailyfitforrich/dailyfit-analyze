# DailyFit 體態分析報告系統

> 專為健身教練設計的體驗課體態評估工具  
> **版本 3.0** — 繁體中文介面 · 角度量測 · 前傾肩角 FSA · 前彎測試

---

## 功能簡介

DailyFit 體態分析報告系統是一款在地運行的健身教練工具，  
適用於健身房體驗課中，教練為學員拍攝姿勢照片後，快速生成體態觀察報告，  
協助解說身體使用狀況、姿勢不平衡傾向，並提供後續建議訓練方向。

### 核心功能
- 上傳四面站姿照片（正面 / 背面 / 左側 / 右側）+ 選填前彎測試
- MediaPipe Pose 自動偵測姿勢地標
- 多項姿勢角度計算（頸椎角、FSA、骨盆傾斜、膝關節角度）
- 圖像自動標注（肩線、髖線、角度線、地標）
- 0–100 姿勢平衡分數
- 前三項主要姿勢觀察摘要
- 個性化訓練建議
- 可下載的 HTML 報告（自帶標注圖像）
- JSON 評估資料存檔

## 安裝與執行

### 環境需求
- Python 3.10 以上
- 建議使用虛擬環境

### 安裝步驟

```bash
# 1. 建立虛擬環境
python -m venv venv
source venv/bin/activate      # Mac / Linux
# venv\Scripts\activate       # Windows

# 2. 安裝依賴套件
pip install -r requirements.txt

# 3. 執行應用程式
streamlit run app.py
```

瀏覽器會自動開啟 `http://localhost:8501`

## 分析指標說明

### 正面 / 背面分析
| 指標 | 說明 | 閾值 |
|------|------|------|
| 肩部高低不對稱 | 比較左右肩部垂直位置差異（以肩寬正規化） | 輕度 3% / 中度 6% / 明顯 10% |
| 骨盆高低不對稱 | 比較左右髖部垂直位置差異（以髖寬正規化） | 同上 |

### 側面分析
| 指標 | 計算方式 | 參考範圍 |
|------|----------|----------|
| 頸椎傾斜角 | 耳→肩連線與鉛直線夾角 | < 10° 中立 / 10–20° 輕度 / > 20° 明顯 |
| 前傾肩角 FSA | 近似 C7→肩峰與鉛直線夾角 | < 52° 正常 / ≥ 52° 圓肩傾向 |
| 骨盆傾斜角 | 軀幹線（肩→髖）與鉛直線夾角（2D 估算） | 0–8° 中立參考區間 |
| 膝關節角度 | 髖→膝→踝 夾角 | > 170° 正常 |
| 頭部前引傾向 | 耳朵相對肩峰的水平偏移比例 | 以軀幹長度正規化 |

### 前彎測試
| 指標 | 說明 |
|------|------|
| 前彎肩部對稱性 | 左右肩部高度差異（以肩寬正規化） |
| 脊柱中線偏移 | 肩中點與髖中點水平偏差評估 |

## 計分系統

- 基準分：100 分
- 輕度問題：扣 5 分
- 中度問題：扣 10 分
- 明顯問題：扣 15 分

| 分數範圍 | 類別 |
|----------|------|
| 85–100 | 姿勢相對平衡 |
| 70–84  | 輕度不平衡傾向 |
| 50–69  | 明顯不平衡傾向 |
| < 50   | 多項可見不平衡傾向 |

> ⚠️ 本分數為健身教練內部參考指標，非醫療評分。

## 已知限制

1. **靜態 2D 分析限制**：所有角度均為靜態照片估算，無法取代動態功能評估
2. **MediaPipe 背面偵測**：背面姿勢地標偵測穩定性較低，建議以正面與側面為主要依據
3. **FSA 計算近似**：C7 使用耳肩關係估算，並非直接骨骼定位
4. **骨盆角度估算**：使用軀幹傾斜作為替代指標，具有較大誤差
5. **光線影響**：照片品質、光線、背景對地標偵測精確度有顯著影響

**本工具不構成任何醫療診斷，所有結果均以「傾向觀察」方式呈現。**

## 照片拍攝建議

- 距離：2–3 公尺
- 全身清晰可見（頭頂至足底）
- 盡量穿著貼身服裝以利地標偵測
- 背景簡單、光線充足
- 直式拍攝
